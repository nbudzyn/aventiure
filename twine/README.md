# Open a Twine `.html` File in Twine
- (Install and) open Twine 2
-  Click "Import from file"
   - Select the file
   - (Choose story to replace)

# Save File After Editting
- In Twine:
  - Click the lower left corner
  - Choose "Publish as file"
  - Select the file
  - Save
- (Commit and Push to GitHub)
